Here are some open source bitcoin web apps I have built:


Feel free to fork any of these projects.


  • SegwitAddress.org - Segwit paper wallet app.
  • Explorer - Simple bitcoin block explorer.
  • BIP39 Code Converter - Simple BIP39 mnemonic code converter.
  • JSWallet - Javascript wallet.
  • SweepKey - Tool for sweeping private keys.
  • UTXOBucket - Experimental wallet that stores UTXO data for only addresses in your wallet.
  • MultiSig Paper Wallet - Multisig tool with Transaction Builder and Offline Key Signer.
  • Drag-Drop - Create private keys by drag and dropping a file or image.
  • Cold Signer - Tool to build and sign transactions offline.
JSWallet and UTXOBucket currently both share the same storage space and cannot be used together without deleting your local browser storage when changing between apps.
